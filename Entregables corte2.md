# PLAN DE TRABAJO Y EVIDENCIAS DEL PROYECTO

**Empresa:** PAF (Purificación y Análisis de Fluidos S.A.S)  
**Área:** Tecnología de la Información (TI)  
**Proyecto:** Centralización y gestión de inventario de hardware y software  

---

## 1. Objetivo del Proyecto

Implementar una base de datos centralizada para el control de activos tecnológicos, permitiendo la gestión de hardware, software y actas de entrega y devolución.

---

## 2. Plan de Trabajo (Basado en el Cronograma)

- Fase 1: Planificación  
  Levantamiento de requerimientos y diseño inicial de la base de datos.

- Fase 2: Análisis  
  Inventario actual de hardware, software y usuarios.

- Fase 3: Diseño y Desarrollo  
  Creación de la base de datos, formularios y módulos de gestión.

- Fase 4: Pruebas  
  Validación del sistema con usuarios, incluyendo trabajadores remotos.

- Fase 5: Implementación  
  Puesta en producción, capacitación y documentación.

- Fase 6: Cierre  
  Evaluación final y entrega del proyecto.

---

## 3. Actividades Realizadas (Evidencias del Desarrollo)

- Creación de formularios para el registro de equipos y usuarios.
- Integración con Word para la generación automática de actas de entrega y devolución.
- Recolección manual de información de equipos (RAM, almacenamiento y procesador).
- Inventario físico del cuarto de TI (tablets, equipos de sonido y otros dispositivos).
- Organización inicial de la base de datos con la información recolectada.
- Estructuración de campos y tablas para control de hardware, software y usuarios.
- Definición de formato estándar para actas de entrega y devolución.
